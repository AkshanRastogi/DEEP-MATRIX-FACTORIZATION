# Deep Matrix Factorization for VCG Compression

This project implements Deep Matrix Factorization techniques for compression and reconstruction of Vectorcardiogram (VCG) signals.

## Features
- Signal compression
- Reconstruction analysis
- PRD evaluation
- Compression Ratio calculation
- Comparison with decomposition methods

## Tools Used
- Python
- NumPy
- Pandas
- Matplotlib
- SciPy
- Kaggle Notebooks

## Results
Performance metrics include:
- PRD (Percent Root Difference)
- Compression Ratio (CR)

## Future Work
- Compare DMF with SVD, DCT, Wavelet methods
- Optimize reconstruction quality
